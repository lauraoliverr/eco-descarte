# Requisitos do Sistema

## Requisitos Funcionais

* RF01 – Cadastrar usuário
* RF02 – Login de usuário
* RF03 – Buscar tipo de resíduo
* RF04 – Exibir forma de descarte
* RF05 – Mostrar pontos de coleta
* RF06 – Favoritar pontos

## Requisitos Não Funcionais

* RNF01 – Sistema responsivo
* RNF02 – Tempo de resposta < 3s
* RNF03 – Autenticação segura
* RNF04 – Acesso via navegador
* RNF05 – Interface simples
